# OOP-CPP-Labs
Репозиторий с лабораторными работами по объектно-ориентированному программированию (ООП) на C++. 

## Описание
Этот репозиторий содержит коды, методички и объяснения для лабораторных работ по курсу ООП. Каждая работа демонстрирует ключевые концепции: классы, наследование, контейнеры STL, файлы, полиморфизм и т.д. Финальный проект (RGR) - простая анимация/мультфильм на C++.

Цель: Практика ООП, STL и базовой графики в Visual Studio или аналогах.

## Структура
- **lab1_classes**: Простые классы.
- **lab2_inheritance**: Наследование и полиморфизм.
- **lab3_files**: Работа с файлами.
- **lab4_vector**: Контейнер vector.
- **lab5_list**: Контейнер list.
- **lab6_deque**: Контейнер deque (очередь).
- **lab7_stack**: Контейнер stack (стек).
- **labs8-11_graphics**: Графика и анимация (Visual Studio).
- **rgr_animation**: Финальный проект - мультфильм.

В каждой папке: 
- Код (.cpp).
- README.md с теорией и объяснением.
