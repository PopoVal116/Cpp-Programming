
# Лабораторная работа №4 — Контейнер std::vector

## Теория
`std::vector<T>` — динамический массив с автоматическим управлением памятью.

**Основные методы:**
- `push_back()`, `pop_back()`
- `size()`, `capacity()`
- `operator[]`, `at()`
- `begin()` / `end()` — итераторы

**Преимущества над обычным массивом:**
- Автоматическое расширение
- Знает свой размер

## Реализовано
- Класс `LibraryReader`
- Ввод N читателей в `vector`
- Фильтрация по первой букве имени (I/i)
- Сортировка пузырьком по ФИО
- Добавление нового читателя с сохранением порядка
