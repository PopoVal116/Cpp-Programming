# Лабораторная работа №2 — Наследование и полиморфизм

## Теория
**Наследование** — создание нового класса на основе существующего.

```cpp
class Derived : public Base { ... }
Полиморфизм — возможность переопределять методы в классах-наследниках.
Виртуальные функции — позволяют вызывать переопределённый метод через указатель/ссылку на базовый класс.
C++virtual void print() const;   // в базовом
void print() const override;  // в наследнике
Порядок вызова конструкторов: сначала базовый → потом наследник.

Базовый класс Cinema (название, телефон, адрес, часы работы, кол-во залов)
Наследник Cinemas — добавляет вместимость зала
