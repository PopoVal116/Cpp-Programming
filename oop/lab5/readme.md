
# Лабораторная работа №5 — Контейнер std::list (двусвязный список)

## Теория
`std::list<T>` — двусвязный список. Каждый элемент содержит указатели на предыдущий и следующий.

**Преимущества:**
- O(1) вставка/удаление в любой позиции
- Нет перераспределения памяти

**Основные методы:**
- `push_back()`, `push_front()`
- `erase(iterator)`
- `begin()`, `end()`

## Реализовано
- Класс `Kvartira` (кол-во комнат, этаж, площадь, адрес)
- Меню картотеки обмена квартир:
  - Вывод всех заявок
  - Добавление новой
  - Поиск подходящего варианта (по комнатам + этажу) → удаление при точном совпадении
