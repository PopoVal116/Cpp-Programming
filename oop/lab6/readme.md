
# Лабораторная работа №6 — Контейнер std::deque и очередь

## Теория
**Очередь** — структура данных с принципом FIFO (First In – First Out).

`std::deque<T>` — double-ended queue, позволяет быстро добавлять/удалять с обоих концов.

**Альтернатива:** `std::queue<T>` — адаптер, использующий `deque` по умолчанию.

**Операции:**
- `push_back()` — добавить в конец (хвост)
- `pop_front()` — удалить из начала (голова)
- `front()`, `back()`

**Применение:** моделирование очередей, буферизация, задачи диспетчеризации.

## Реализовано
- Класс `Airport` (рейс, цена билета, мест, продано)
- Чтение N рейсов из файла
- Подсчёт выручки
- Разделение на две очереди: выше и ниже среднего
